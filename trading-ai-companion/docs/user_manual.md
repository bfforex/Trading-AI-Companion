\## Introduction



The Trading AI Companion is an intelligent trading assistant that leverages multiple AI backends to provide comprehensive support for human traders. It automatically detects available local models in Ollama and integrates seamlessly with MetaTrader 5.



\## Installation



\### Prerequisites

\- Python 3.8 or higher

\- Ollama installed and running

\- MetaTrader 5 installed

\- Git (for cloning the repository)
